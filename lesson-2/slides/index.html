Absolute URL

http://codeparkhouston.com/materials/lesson-2/slides/assets/images/glasses.jpg
  • Starts with http:// or https://
  • Has full domain
  • Will absolutely go to where the URL points no matter where you go to the link from *

* Unless some hackery is going on ;)

Relative URL

http://codeparkhouston.com/materials/lesson-2/slides/assets/images/glasses.jpg
  • Does not start with http:// or https://
  • Does not have domain
  • Where the URL points to is relative to where you are accessing it from
